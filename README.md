# 🛒 Walmart Sales Dashboard | Power BI

## 🚀 Project Overview
This project presents an interactive **Walmart Sales Dashboard** built using Power BI to analyze sales performance, product trends, and profitability.

The dashboard provides a comprehensive view of:
- Sales performance  
- Product and category trends  
- Customer insights  
- Profitability analysis  

---

## 📊 Dashboard Highlights

### 🔹 Key KPIs

- 💰 **Total Sales:** 725K+  
- 📦 **Total Quantity Sold:** 12K+  
- 📈 **Total Profit:** 108K+  
- 🚚 **Average Delivery Time:** 3.93 days  

---

## 📈 Key Visualizations

- 📅 **Sales by Month & Quarter** (trend analysis)  
- 🏙️ **Sales by City**  
- 📦 **Top 5 Products**  
- 🧾 **Top 5 Categories**  
- 👥 **Top 5 Customers**  
- 📊 **Category-wise Sales, Quantity, and Profit Table**  

---

## 📌 Key Insights

- A few products contribute significantly to total sales  
- Certain categories generate high sales but low profit  
- Sales vary significantly across cities  
- Seasonal trends impact monthly performance  
- Some categories show negative or low profitability  

---

## 💡 Business Recommendations

- Focus on **high-performing products and categories**  
- Review **low-profit or loss-making categories**  
- Optimize **inventory and supply chain**  
- Target high-value customers for retention  
- Use seasonal trends for better sales planning  

---

## 🛠️ Tools & Technologies

- Power BI  
- DAX (Data Analysis Expressions)  
- Data Visualization  
- Dashboard Design
